## 🎨 Diseño sugerido

### Paleta de colores
- **Fondo**: #4A3218 (marrón caramel)
- **Texto**: Blanco o beige claro
- **Icono**: 🍰 o 🧁 (pastel/cupcake)

### Estilo recomendado
- **Minimalista** y profesional
- **Legible** en pequeños tamaños
- **Consistente** con la marca "Dulce & Masa"

## 🛠️ Herramientas para crear íconos

### Opción 1: Online (fácil)
1. **Canva** - Plantillas PWA
2. **Favicon.io** - Generador automático
3. **PWA Asset Generator** - Sube una imagen y genera todos los tamaños

### Opción 2: Diseño profesional
1. **Adobe Illustrator** o **Figma**
2. **Photoshop** para optimización
3. **Exportar** en PNG con transparencia

### Opción 3: Automático desde una imagen
```bash
# Usar pwa-asset-generator
npm install -g pwa-asset-generator
pwa-asset-generator icon-input.png ./output/
```

## 📋 Checklist de íconos

- [ ] **192x192** - Principal para instalación

## 🎯 Proceso rápido

1. **Crea un diseño base** en 512x512
2. **Exporta** a todos los tamaños requeridos
3. **Nombra** los archivos correctamente
4. **Coloca** en la raíz del proyecto
5. **Testea** la instalación PWA

## 🚀 Una vez listo

Cuando tengas los íconos:
1. **Sube** todos los archivos a GitHub
2. **Activa** GitHub Pages
3. **Testea** la instalación PWA
4. **Verifica** que aparezca el ícono correctamente
